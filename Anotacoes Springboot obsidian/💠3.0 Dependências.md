### 💠 O que são dependências?

Pensa assim: você vai construir uma casa. Você poderia fabricar cada tijolo do zero, misturar o cimento você mesmo, fundir o ferro das vigas... mas isso seria absurdo. Você **compra materiais prontos** de fornecedores especializados e usa na sua obra.

Dependências são exatamente isso — **código pronto, feito por outras pessoas**, que você declara no seu projeto e usa sem precisar escrever do zero. O [[💠 2.0 Maven]] (aquele `pom.xml` do seu projeto) é o "fornecedor" que vai buscar e entregar esses materiais automaticamente.


## 💠 As dependências desse projeto

- **Spring Web** É o que transforma seu projeto num servidor HTTP. Sem ele, sua aplicação Spring não consegue receber requisições da internet. Ele é quem vai pegar uma chamada como `GET /usuarios` e encaminhar para o código certo. É a porta de entrada do seu sistema.

- **Spring Data JPA** É a ponte entre seu código Java e o banco de dados. Sem ele, você teria que escrever SQL na mão para tudo — buscar, salvar, atualizar, deletar. Com ele, você escreve Java e ele se encarrega de gerar o SQL por baixo dos panos. JPA é a especificação, e o Hibernate (que vem junto) é quem faz o trabalho pesado.

- **PostgreSQL Driver** É o tradutor entre o Java e o PostgreSQL especificamente. O Spring Data JPA sabe falar "banco de dados genérico", mas precisa de alguém que saiba falar "PostgreSQL" de verdade. Esse driver é esse alguém. Se você fosse usar MySQL, trocaria por um driver MySQL — a lógica é a mesma.

- **Flyway Migration** Imagina que você e seu colega estão desenvolvendo juntos, cada um com seu banco de dados local. Você cria uma tabela nova. Como você garante que o banco dele também vai ter essa tabela? Você poderia mandar um SQL por WhatsApp... mas isso escala mal. O Flyway resolve isso — você cria arquivos de migração versionados (tipo `V1__criar_tabela_usuario.sql`) e ele garante que todo banco de dados do projeto, seja na sua máquina, na do seu colega ou no servidor de produção, esteja sempre na mesma versão. É um controle de versão pro seu banco.
