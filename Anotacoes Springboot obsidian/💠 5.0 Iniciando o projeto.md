## 💠 Implementando um link para o nosso banco de dados

Primeiramente, vamos acessar um arquivo de configuração do nosso projeto, para fazê-lo reconhecer nosso banco de dados.

Esse arquivo é o `application.properties`, que se localiza em:

 `pasta_do_projeto > src > main > resources > application.properties`

Devemos inserir as seguintes linhas nesse arquivo:

``` 
spring.datasource.url = jdbc:postgresql://localhost:5432/product-api
spring.datasource.username = postgres
spring.datasource.password = 1234
```

Com isso, estamos inserindo link e informações a respeito do banco de dados que estaremos utilizando no nosso projeto.

## 💠 Criando o banco de dados no pgAdmin

O pgAdmin4 é uma interface para criar bancos de dados PostgreSQL.

Para iniciar nosso banco de dados, devemos ir em:

`Servers > PostgreSQL > Databases > Create > Database`

![[{3443B89A-AF91-4334-AFCF-4FADEE310215}.png]]

Depois disso, basta dar um nome à base de dados, igual ao que foi definido no arquivo `application.properties`, e clicar em salvar. A base de dados será criada instantaneamente.
