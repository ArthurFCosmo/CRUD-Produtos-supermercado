Para criar o Controller, que nada mais é do que o receptor das [[💠 7.1 Requisições HTTP]], devemos primeiro criar um pacote:

> `right click na pasta do projeto > new > package > com.cosmodev.crud1.controllers`

Dentro desse pacote, criaremos uma classe:

> `right click no pacote > new > Java class > ProductController`

---

Nessa classe, adicionaremos duas anotações:

```java
package com.cosmodev.crud1.controllers;  
  
import org.springframework.web.bind.annotation.RequestMapping;  
import org.springframework.web.bind.annotation.RestController;  
  
@RestController  
@RequestMapping("/products")  
public class ProductController {  
}
```

⚠️ **ATENÇÃO ÀS IMPORTAÇÕES**

- **@RestController** - Informa ao Spring que a classe a seguir se trata de uma classe de Controller, uma classe controladora.

- **@RequestMapping("/products")** - Informa ao Spring que sempre que houver uma requisição http com o endereço `/products`, ele deve se dirigir a esse controlador para utilizar de seus recursos.
